Generator-m-ionic Configuração Do Ambiente no Windows

Por Wallace Coelho, sex 09 dezembro 2016, na categoria Desenvolvimento

Configuração, Cordova, Ionic, Mobile, Windows

Quer iniciar o desenvolvimento de apps Híbridos e não sabe por onde começar? Vou ensinar neste tutorial como configurar o ambiente completo para que você possa dar os primeiros passos.

Instalação do Node JS

Primeiramente, instale o Node JS em nodejs.org, baixe a versão LTS (atualmente v6.9.2)

Instale a última versão do RubyInstaller, atualmente a versão LTS é a 2.2.6 Marque a opção "Add Ruby Executables to your PATH"

Instalação do Ruby (Para usar os recursos do SASS)

Finalize a instalação do Ruby

Instalação dos Módulos do Node

Abra o prompt de comando em qualquer lugar e rode

npm install -g cordova bower ionic gulp yo generator-m-ionic

Este comando instalará globalmente (para todo o sistema) as Dependências

  • Cordova: Plataforma para apps Híbridos
  • Bower: Gerenciador de Dependências do Frontend
  • Ionic: Framework Mobile
  • Gulp: Automatização de Processos
  • Yoman: Ferramenta para download de Scaffolds
  • Generator-m-Ionic: Scaffold para Ionic

Configurando o Android

Instalando o Android SDK

Vá na página de download do Android e na seção Obtenha apenas as ferramentas de linha de comando baixe as ferramentas do Windows. Ao término do Download abra o Zip e extraia o conteúdo da pasta tools para C:\android (ou qualquer outro diretório que preferir).

Precisamos adicionar este diretório no PATH do Windows. Clique com o botão direito em Meu Computador, vá em propriedades, Configurações Avançadas do Sistema e e clique em variáveis de Ambiente.

Clique em Novo (Em variáveis de Ambiente) e crie a variável ANDROID_HOME com o caminho que colocamos o android (C:\android)

Selecione agora a variável PATH e clique em editar. Nela colocaremos ;%ANDROID_HOME%\tools

IMPORTANTE: Não adicione espaços e use ponto e vírgula (;) para separar.

Clique em ok para fechar tudo

Instalando o Java JDK

Baixe o JDK no site oficial. Clique em Accept e baixe de acordo com sua arquitetura (x32/x86 ou x64)

Instale o Java normalmente e, ao término, adicione uma nova variável de ambiente.

Nome = JAVA_HOME Valor = C:\Program Files\Java\jdk1.8.0_111

Obs: verifique qual o nome da sua pasta, pois a versão pode ser diferente de jdk1.8.0_111.

Adicione ao PATH: ;%JAVA_HOME%\bin

Obs 2: Olhe na sessão Instalando o Android SDK para saber como adicionar uma variável de Ambiente.

Instalando as Plataformas do Android

Abra o prompt e digite android deverá abrir um gerenciador.

  • Em tools, selecione todas.
  • Escolha uma plataforma - Irei instalar todos os componentes da versão 6.0 (API23)
  • Em Extras, selecione Google USB Driver

Clique em install e aguarde (Este processo pode demorar um pouco).

Quando tudo tiver baixado, feche o gerenciador.

Iniciando nosso projeto

Crie uma nova pasta, abra o prompt na pasta criada e rode o comando:

yo m-ionic

Este comando irá gerar uma série de perguntas para dar o 'start' em nosso projeto. Para uma descrição completa de cada pergunta, aconselho a olhar na documentação oficial.

Obs: Aconselho a instalar com o SASS (se você já instalou o Ruby), localforage, Device e SplashScreen no mínimo.

Com isso já temos um ambiente pronto para desenvolver aplicativos e rodar no Android.
